lift = normalized hit rate = batch_size * hit_rate

analysis:
   completely random NHR would be batch_size*(1/batch_size) = 1
   so NHR > 1 +- std error shows a model is doing better than random.

   we would expect:
      healthy learning: 1.9 - 2.9
      strongly optimized: 3.2 - 4.8
      potential data leakage: >= 15

   An NHR > 1 but less than 1.5-ish suggests that the model is
   not quite capturing personalization yet and so more features
   may need to be enabled.
